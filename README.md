# Shubhashree R - Portfolio

A modern, responsive portfolio website showcasing professional experience, skills, and projects.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern Styling**: Gradient backgrounds and smooth animations
- **Professional Layout**: Organized sections for About, Skills, Projects, and Contact
- **Interactive Elements**: Hover effects and smooth transitions

## File Structure

```
portofolio 2/
├── a.html          # Main portfolio website
└── README.md       # This file
```

## Getting Started

1. Open `a.html` in a web browser to view the portfolio
2. The page is self-contained with all CSS included inline

## Technologies Used

- HTML5
- CSS3 (with gradients, flexbox, and animations)

## Customization

You can customize the portfolio by editing:
- **Name and Title**: Update the header section
- **Colors**: Modify the gradient colors (`#667eea`, `#764ba2`)
- **Content**: Add your projects, skills, and experience
- **Links**: Update contact and social media links

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
